# saltstack-web-javascript-chart

## Description
A small project to deminstrate using
client side scripting for metrics. Salt was used to build the apache server.

## Tech stack
- salt
- jquary 3.5
- chart.js 2.9.4

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
